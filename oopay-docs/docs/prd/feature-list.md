## 2. 功能详情

### 2.1 商户中心 (Merchant)

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| M-01 | 商户入驻 | 提交资料，等待审核 | P0 |
| M-02 | 商户审核 | 运营人员审核入驻申请 | P0 |
| M-03 | 商户信息 | 查看/修改商户基本信息 | P0 |
| M-04 | API 密钥 | 生成/重置 API 密钥 | P0 |
| M-05 | IP 白名单 | 配置可访问 IP | P1 |
| M-06 | 回调地址 | 配置异步通知地址 | P0 |
| M-07 | 通道配置 | 绑定可用支付通道 | P0 |
| M-08 | 费率查看 | 查看各通道费率 | P1 |

### 2.2 支付核心 (Payment)

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| P-01 | 统一下单 | 创建支付订单 | P0 |
| P-02 | 订单查询 | 查询订单状态 | P0 |
| P-03 | 订单关闭 | 关闭未支付订单 | P1 |
| P-04 | 支付回调 | 接收并处理通道回调 | P0 |
| P-05 | 退款申请 | 发起退款请求 | P1 |
| P-06 | 退款查询 | 查询退款状态 | P1 |
| P-07 | 批量查询 | 批量查询订单状态 | P2 |

### 2.3 账户资金 (Account)

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| A-01 | 余额查询 | 查询账户余额 | P0 |
| A-02 | 资金冻结 | 冻结指定金额 | P0 |
| A-03 | 资金解冻 | 解冻已冻结金额 | P0 |
| A-04 | 流水查询 | 查询资金流水 | P1 |
| A-05 | 流水导出 | 导出流水记录 | P2 |
| A-06 | 限额配置 | 配置日限额/单笔限额 | P1 |

### 2.4 对账中心 (Reconcile)

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| R-01 | 实时对账 | 分钟级对账检查 | P1 |
| R-02 | 日终对账 | 每日全量对账 | P1 |
| R-03 | 差异查看 | 查看对账差异 | P1 |
| R-04 | 差异处理 | 处理对账差异 | P1 |
| R-05 | 对账报表 | 生成对账报表 | P2 |

### 2.5 风控管理 (Risk)

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| K-01 | 规则配置 | 配置风控规则 | P1 |
| K-02 | 拦截记录 | 查看拦截记录 | P1 |
| K-03 | 风控等级 | 设置商户风控等级 | P1 |
| K-04 | 大额确认 | TG Bot 二次确认 | P1 |

### 2.6 运营管理 (Ops)

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| O-01 | 通道管理 | 配置支付通道 | P0 |
| O-02 | 通道监控 | 监控通道健康状态 | P1 |
| O-03 | 费率配置 | 配置通道费率 | P0 |
| O-04 | 订单监控 | 实时监控订单异常 | P1 |

## 3. 功能依赖关系

```
商户入驻 ──► 商户审核 ──► 通道配置 ──► 发起支付 ──► 订单管理
                │                              │
                ▼                              ▼
           API 密钥配置 ◄────────────────── 账户资金
                │
                ▼
           风控配置 ──► 支付风控拦截
```
