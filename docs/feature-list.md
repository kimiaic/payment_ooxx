# OOPay 系统功能清单

## 一、运营后台 (oopay-manage)

### 1. 仪表盘
| 二级功能 | 功能说明 |
|---------|---------|
| 数据概览 | 今日订单数、交易额、成功率、新增商户 |
| 额度监控 | 各厂商可用额度进度条、预警状态 |
| 实时动态 | 最新交易流水、成功/失败状态 |
| 快捷入口 | 厂商充值、商户审核、异常订单处理 |

### 2. 厂商管理
| 二级功能 | 功能说明 |
|---------|---------|
| 厂商列表 | 查看所有上游厂商、额度状态、今日数据 |
| 新增厂商 | 录入厂商信息、USDT充值地址、初始额度 |
| 额度管理 | 充值、扣减、调整额度，查看变动记录 |
| 额度预警 | 设置预警阈值、接收TG告警、预警历史 |
| 厂商详情 | 基本信息、额度概览、今日统计、操作日志 |

### 3. 通道管理
| 二级功能 | 功能说明 |
|---------|---------|
| 通道列表 | 查看所有上游通道、所属厂商、费率、限额、状态 |
| 新增通道 | 配置通道编码、支付类型、费率、单笔/日限额 |
| 通道配置 | 修改费率、调整限额、上线/下线/维护状态切换 |
| 通道监控 | 成功率统计、响应时间、今日订单量 |

### 4. 产品管理
| 二级功能 | 功能说明 |
|---------|---------|
| 产品列表 | 查看所有支付产品、编码、类型、默认费率、状态 |
| 新增产品 | 创建产品编码、名称、选择支付类型、设置默认费率 |
| 路由配置 | 绑定上游通道、设置权重比例、优先级、备用通道 |
| 高级规则 | 设置金额筛选规则（如单笔>5000优先某通道） |
| 产品授权 | 查看哪些商户已授权此产品 |

### 5. 商户管理
| 二级功能 | 功能说明 |
|---------|---------|
| 商户列表 | 查看所有商户、余额、状态、注册时间 |
| 邀请码生成 | 生成邀请码、设置有效期、查看使用记录 |
| 商户审核 | 查看待审核商户、通过/拒绝、设置审核备注 |
| 产品授权 | 为商户开通支付产品、设置独立费率 |
| IP白名单 | 配置商户允许下单的服务器IP列表 |
| 商户详情 | 基本信息、授权产品、余额、订单统计、操作日志 |
| 商户冻结 | 冻结/解冻商户账户、设置冻结原因 |

### 6. 订单管理
| 二级功能 | 功能说明 |
|---------|---------|
| 订单查询 | 按订单号、商户、通道、状态、时间范围筛选 |
| 订单详情 | 完整订单信息、三方状态对比（平台/上游/商户） |
| 手动补单 | 标记订单为已支付、扣减额度、通知商户 |
| 强制关单 | 关闭异常订单、释放冻结额度 |
| 查询上游 | 主动查询上游订单真实状态 |
| 差异订单 | 查看对账差异列表、处理状态跟踪 |

### 7. 资金管理
| 二级功能 | 功能说明 |
|---------|---------|
| 资金流水 | 查看所有资金变动记录、按类型/账户筛选 |
| 厂商充值 | 为上游厂商增加额度、填写充值金额、备注 |
| 商户结算 | 为商户处理提现申请、批准/拒绝 |
| 手动调账 | 调整商户或厂商余额、填写调账原因 |
| 收益统计 | 平台日收益、周收益、月收益统计 |
| 对账报表 | 厂商对账、商户对账、差异报表 |

### 8. 风控管理
| 二级功能 | 功能说明 |
|---------|---------|
| IP黑名单 | 添加/删除黑名单IP、查看拦截记录 |
| 商户黑名单 | 将商户加入黑名单、查看关联订单 |
| 风控规则 | 配置风控策略（金额限制、频次限制） |
| 风险订单 | 查看风控拦截订单、处理记录 |
| 风控日志 | 风控决策记录、规则命中统计 |

### 9. 系统配置
| 二级功能 | 功能说明 |
|---------|---------|
| 基础配置 | 订单超时时间、回调重试次数、汇率更新间隔 |
| 汇率配置 | OKX API配置、默认汇差、汇率异常处理 |
| 通知配置 | TG Bot Token、通知群组绑定、消息模板 |
| 权限管理 | 运营账号管理、角色配置、权限分配 |
| 操作日志 | 后台所有操作审计、按时间/操作人/类型筛选 |

## 二、商户系统 (oopay-merchant)

### 1. 首页
| 二级功能 | 功能说明 |
|---------|---------|
| 账户概览 | 可用余额、冻结金额、今日收入卡片 |
| 今日统计 | 订单总数、成功数、失败数、成功率 |
| 快捷入口 | 查看订单、下载对账单、API文档、联系客服 |
| 系统公告 | 平台通知、维护公告、费率调整通知 |

### 2. 基础信息
| 二级功能 | 功能说明 |
|---------|---------|
| 商户资料 | 查看/修改商户名称、联系人、手机号、邮箱 |
| 结算地址 | 配置USDT结算地址（TRC20）、修改地址 |
| 密码管理 | 修改登录密码、设置/修改支付密码 |
| 安全设置 | 二次验证开关、安全问题设置 |

### 3. 支付通道
| 二级功能 | 功能说明 |
|---------|---------|
| 通道列表 | 查看平台授权的支付产品（代码/名称/费率/状态） |
| 通道详情 | 费率说明、限额说明、接入参数 |
| 费率对比 | 与平台默认费率对比、优惠展示 |

### 4. 代付通道（预留）
| 二级功能 | 功能说明 |
|---------|---------|
| 代付列表 | 查看可用的代付通道、费率、限额 |
| 默认设置 | 设置默认代付通道 |
| 代付记录 | 历史代付订单查询（后续版本） |

### 5. 安全中心
| 二级功能 | 功能说明 |
|---------|---------|
| API密钥 | 查看AppID、API Key、Secret Key、重置密钥 |
| IP白名单 | 添加/删除下单服务器IP、查看IP拦截记录 |
| 登录日志 | 近期登录记录（时间/IP/地点/结果） |
| 安全提醒 | 异常登录提醒、密码修改提醒 |

### 6. 资金流水
| 二级功能 | 功能说明 |
|---------|---------|
| 余额概览 | 当前余额、冻结金额、本月收入/支出 |
| 收入记录 | 订单收款明细、筛选查询 |
| 支出记录 | 手续费、提现扣款明细 |
| 流水明细 | 完整的资金变动历史、导出Excel |
| 余额预警 | 设置余额不足提醒阈值 |

### 7. 订单管理
| 二级功能 | 功能说明 |
|---------|---------|
| 订单查询 | 按平台订单号、商户订单号、状态、时间筛选 |
| 订单列表 | 查看所有订单（金额、通道、状态、时间） |
| 订单详情 | 订单完整信息、支付链接（待支付订单） |
| 订单导出 | 导出订单数据（Excel格式） |

### 8. 结算管理
| 二级功能 | 功能说明 |
|---------|---------|
| 申请结算 | 发起USDT提现申请、填写金额、确认地址 |
| 结算记录 | 历史结算列表（金额/时间/状态/交易哈希） |
| 结算状态 | 待审核/处理中/已完成/失败 |
| 结算统计 | 本月结算金额、结算次数 |

### 9. 对账报表
| 二级功能 | 功能说明 |
|---------|---------|
| 日报 | 每日订单汇总（订单数/金额/成功率） |
| 月报 | 月度统计（交易趋势图、收入统计） |
| 通道分析 | 各支付通道成功率对比、交易量占比 |
| 报表导出 | 下载日报/月报（PDF/Excel） |

### 10. 对接信息
| 二级功能 | 功能说明 |
|---------|---------|
| API文档 | 接口说明文档（请求/响应/错误码） |
| SDK下载 | Java/PHP/Python SDK下载 |
| 接入指南 | 快速接入教程、常见问题 |
| 调试工具 | 在线签名生成器、订单查询测试工具 |
| 示例代码 | 各语言Demo代码 |

## 三、支付网关 (oopay-pay)

### 1. 接入网关
| 二级功能 | 功能说明 |
|---------|---------|
| 请求鉴权 | API Key校验、签名验证、IP白名单校验 |
| 请求限流 | 商户级限流、接口级限流、突发流量控制 |
| 参数校验 | 请求参数格式校验、必填校验、业务规则校验 |
| 日志记录 | 请求日志、响应日志、异常日志 |

### 2. 订单处理
| 二级功能 | 功能说明 |
|---------|---------|
| 订单创建 | 生成平台订单号、保存订单信息、冻结额度 |
| 订单查询 | 按订单号查询订单状态 |
| 订单关闭 | 超时关单、手动关单、释放额度 |
| 幂等控制 | 防止重复下单、重复处理 |

### 3. 路由引擎
| 二级功能 | 功能说明 |
|---------|---------|
| 通道筛选 | 按在线状态、额度、限额筛选可用通道 |
| 权重分配 | 按配置权重随机选择主通道 |
| 失败切换 | 主通道失败自动尝试备用通道 |
| 熔断机制 | 连续失败自动熔断、健康探测恢复 |
| 路由日志 | 记录每次路由决策结果 |

### 4. 上游适配
| 二级功能 | 功能说明 |
|---------|---------|
| 通道接口 | 对接各上游的支付接口 |
| 参数转换 | 将平台参数转换为上流格式 |
| 响应处理 | 处理上游返回的支付链接/二维码 |
| 超时处理 | 上游接口超时处理、降级策略 |

### 5. 回调处理
| 二级功能 | 功能说明 |
|---------|---------|
| 接收回调 | 接收上游支付结果通知 |
| 回调验证 | 验证回调签名、防篡改 |
| 回调幂等 | 防止同一订单重复处理 |
| 状态更新 | 更新订单状态、扣减额度、计算利润 |
| 商户通知 | 异步通知商户支付结果 |
| 通知重试 | 失败重试机制（1/5/15分钟） |

## 四、定时任务 (oopay-task)

### 1. 对账任务
| 二级功能 | 功能说明 |
|---------|---------|
| 兜底查询 | 每5分钟查询待支付订单状态 |
| 超时关单 | 关闭超过2小时未支付订单 |
| 差异检测 | 比对平台与上游订单状态 |
| 差异告警 | 推送差异订单到运营群 |

### 2. 结算任务
| 二级功能 | 功能说明 |
|---------|---------|
| 自动结算 | 处理商户自动结算申请 |
| 收益归集 | 汇总平台收益到指定账户 |
| 结算对账 | 核对结算记录与资金变动 |

### 3. 数据任务
| 二级功能 | 功能说明 |
|---------|---------|
| 账单生成 | 每日凌晨生成商户账单 |
| 统计汇总 | 计算日/周/月统计数据 |
| 数据清理 | 清理过期日志、归档历史数据 |
| 数据备份 | 数据库备份、关键数据导出 |

### 4. 监控任务
| 二级功能 | 功能说明 |
|---------|---------|
| 额度巡检 | 检查厂商额度、触发预警 |
| 通道巡检 | 探测上游通道健康状态 |
| 一致性巡检 | 检查数据一致性异常 |
| 资金平衡 | 核对各账户余额平衡 |

### 5. TG Bot任务
| 二级功能 | 功能说明 |
|---------|---------|
| 额度预警推送 | 厂商额度不足时推送告警 |
| 账单推送 | 每日推送商户账单到合作群 |
| 回调失败推送 | 回调商户失败时推送告警 |
| 异常订单推送 | 对账差异、通道熔断推送 |
| 指令处理 | 处理Bot指令、执行操作 |
| 指令日志 | 记录所有指令执行记录 |

## 五、基础支撑

### 1. 核心服务 (oopay-service)
| 二级功能 | 功能说明 |
|---------|---------|
| 上游服务 | 厂商管理、额度管理、通道管理 |
| 商户服务 | 商户管理、应用管理、产品授权 |
| 订单服务 | 订单创建、状态流转、查询统计 |
| 账户服务 | 余额管理、流水记录、结算处理 |
| 风控服务 | 规则引擎、风险评估、黑名单 |
| 对账服务 | 订单对账、资金对账、差异处理 |

### 2. 核心实体 (oopay-core)
| 二级功能 | 功能说明 |
|---------|---------|
| 实体定义 | 订单、商户、厂商、账户等实体 |
| 接口定义 | Dubbo服务接口 |
| 公共常量 | 状态码、枚举类型 |
| 工具类 | 日期、金额、加解密、签名 |
| 异常体系 | 业务异常定义 |

### 3. 商户SDK (oopay-z-api-base)
| 二级功能 | 功能说明 |
|---------|---------|
| SDK封装 | Java/PHP/Python SDK |
| 签名工具 | 请求签名生成 |
| HTTP客户端 | 带重试、超时的HTTP请求 |
| 示例项目 | Demo代码、快速接入示例 |
