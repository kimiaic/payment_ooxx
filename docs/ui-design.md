# OOPay UI 界面设计文档

## 设计规范

### 配色
- 主色：#1890ff (蓝色)
- 成功：#52c41a (绿色)
- 警告：#faad14 (橙色)
- 错误：#f5222d (红色)
- 背景：#f0f2f5
- 文字：#262626 (主) / #595959 (次)

### 布局
- 左侧固定导航 (200px)
- 顶部标题栏 (64px)
- 内容区自适应
- 卡片圆角：8px
- 间距：16px/24px
